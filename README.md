# paledit
仙剑奇侠传DOS版存档修改器

此工具为仙剑奇侠传DOS版存档修改器

适用于移植自仙剑DOS版的各个平台的存档修改

PPC，PC，S60，PSP等

程序使用c#，基于.net CF库，任何支持.net CF的平台都可运行此工具。

---


trunk下的代码为pc/ppc通用的代码

branches下为魅族 M8专用
